---
title: Set up your Ory development environment
---

Set up your Ory development environment by following the steps below. They take you from a 
new Ory account to a working environment for building your application's authentication and 
authorization on Ory Network. Complete them in order; each one assumes you've finished the last.

1. Create your Ory account
2. Install the Ory CLI
3. Create an Ory project
4. Install the Ory SDK
5. Set up Ory Tunnel
