---
id: mfa
title: This should be the Mfa UI
---

You ended up here because you didn't set the following configuration value:

```yaml title="path/to/kratos/config.yml
selfservice:
  flows:
    mfa:
      ui_url: http://my-app.com/mfa
```

You can set this configuration value using environment variable `SELFSERVICE_FLOWS_MFA_UI_URL` as well!
