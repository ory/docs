# Setup links for directory sync

The Setup link is used to create a shareable link that can be used to create an Directory Sync without exposing any sensitive
information. Let's go through the steps to create a new Setup link.

## Create Setup link

You can create a new setup link for Directory Sync from the Connections page using `New Setup Link` button

![dsync connections page](../_assets/admin-portal/setup-links/dsync/dsync-connections-page.png)

Another way is to go to the `Setup Links` section under `Directory Sync` menu and Click on Create New/New Setup Link button

![dsync setup link list empty](../_assets/admin-portal/setup-links/dsync/dsync-setup-link-list-empty.png)

The create Setup Link for Directory Sync page looks like below

![dsync setup link create page](../_assets/admin-portal/setup-links/dsync/dsync-setup-link-create-page.png)

Once the link is created you can see the the link info section at the bottom

![setup link created](../_assets/admin-portal/setup-links/dsync/setup-link-created.png)

You can copy the link by clicking on the `Copy` button

![setup link copied](../_assets/admin-portal/setup-links/dsync/setup-link-copied.png)

### Regenerate link

You can regenerate the link by clicking on `Regenerate` button

![setup link regenerate init](../_assets/admin-portal/setup-links/dsync/setup-link-regenerate-init.png)

Once you confirm Regenerate action you will see the new setup link in the link info section

![setup link regenerated](../_assets/admin-portal/setup-links/dsync/setup-link-regenerated.png)

### Actions from setup link list page

You can perform `ACTIONS` like `Copy`, `View`, `Regenerate` & `Delete` from this page

![setup link list page](../_assets/admin-portal/setup-links/dsync/setup-link-list-page.png)

### Setup link page

When you open the setup link you will see the following

![setup link page dsync](../_assets/admin-portal/setup-links/dsync/setup-link-page-dsync.png)

### Create Directory Sync connection

You can open create Directory Sync by clicking on `New Directory` button & filling the correct details

![dsync setup link created](../_assets/admin-portal/setup-links/dsync/dsync-setup-link-created.png)

After creation of the directory sync you will see the info section

![dsync created info](../_assets/admin-portal/setup-links/dsync/dsync-created-info.png)
