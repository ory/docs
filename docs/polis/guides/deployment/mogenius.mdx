# mogenius

This guide will walk you through the configuration and deployment of Ory Polis on mogenius. mogenius is the easiest way for
developers to run any application in the cloud. They have a template for Ory Polis, so that you can deploy it on their fully
automated cloud infrastructure in minutes.

If you donâ€™t have a mogenius account, create one [here](https://studio.mogenius.com/user/registration). They have an extensive
[guide](https://docs.mogenius.com/overview/quickstart) to get you started.

## Deploy Ory Polis

- Head over to [mogenius](https://docs.mogenius.com/templates-and-services/various/saml-jackson) and follow the instructions to
  deploy Ory Polis as a docker container.
- You should deploy the latest version from [Docker Hub](https://hub.docker.com/r/boxyhq/jackson/tags) and please keep your image
  up to date for additional security.

### Verify deployment

You can visit the path `/api/hello` on your app to see if Ory Polis was started. If the installation was successful, you'll see
the following output on the screen.

```javascript
{
  name: "Jules Winnfield"
}
```
