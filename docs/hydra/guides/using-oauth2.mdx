Use open source & battle-tested libraries to consume OAuth2 and OpenID Connect:

- Node.js
  - [node-openid-client](https://github.com/panva/node-openid-client)
  - [passport](http://www.passportjs.org/)
  - [simple-oauth2](https://github.com/lelylan/simple-oauth2)
- [Golang](https://github.com/golang/oauth2)
- [PHP](https://github.com/thephpleague/oauth2-client)
- [Java](https://spring.io/projects/spring-security)
- [Objective C](https://github.com/openid/AppAuth-iOS)
- [Erlang](https://github.com/exograd/erl-oauth2c)
- [Elixir](https://github.com/scrogson/oauth2)
- [Ruby](https://github.com/nov/rack-oauth2)
- [Rust](https://github.com/ramosbugs/oauth2-rs)
- [Scala](https://github.com/ocadotechnology/sttp-oauth2)
- [Swift](https://github.com/openid/AppAuth-iOS)
- [Python](https://github.com/lepture/authlib)

For a more client libraries go [here](https://oauth.net/code/).
