[Clearbit](https://clearbit.com) (now HubSpot Breeze Intelligence) —
person and company data enrichment.

## How it works

Ory Action on `registration.after` (async) → handler calls Clearbit /
Breeze Intelligence Person API with the user's email → PATCHes
returned enrichment to `metadata_public.clearbit` on the identity.

## Notable

- New deployments use HubSpot Breeze Intelligence over
  legacy Clearbit endpoints.
- Best-effort enrichment — silently no-op on 404 misses.
- DSAR delete API available — wire to your right-to-be-forgotten
  flow.
