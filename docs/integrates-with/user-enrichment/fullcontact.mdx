[FullContact](https://www.fullcontact.com) — person/company data
enrichment.

## How it works

Ory Action on `registration.after` (async) → handler calls
`POST https://api.fullcontact.com/v3/person.enrich` → PATCHes returned
enrichment to `metadata_public.fullcontact` on the Ory identity.

## Notable

- Best-effort enrichment — silently no-op on 404 misses.
- FullContact's multi-field resolution improves hit rate at the cost
  of more credits used.
- DSAR delete API available.
