[Microsoft Sentinel](https://learn.microsoft.com/en-us/azure/sentinel/)
— cloud SIEM + SOAR on Azure. Ingest Ory identity events via the Log
Analytics Data Collector API (or newer Logs Ingestion API + DCRs).

## How it works

Ory Action → handler signs request with Azure HMAC-SHA256 → POSTs to
`https://<workspace-id>.ods.opinsights.azure.com/api/logs` under custom
log type `OryEvents_CL`. KQL analytics rules in Sentinel query
`OryEvents_CL` for detection logic; Logic Apps playbooks automate
response.

## Notable

- Data Collector API is being deprecated in favor of the **Logs
  Ingestion API + DCRs**; new deployments use the newer API.
- Alternative no-code path: Logic App as intermediary — receives the
  Ory Action webhook and posts to Sentinel.
