[Splunk](https://www.splunk.com) — log management and SIEM. Ingest Ory
authentication events directly via Splunk's HTTP Event Collector (HEC);
no webhook handler needed.

## How it works

Ory Action POSTs directly to
`https://<splunk-host>:8088/services/collector/event` with
`Authorization: Splunk {$HEC_TOKEN}` header. Body Jsonnet wraps the
event in Splunk's HEC envelope.

## Notable

- For Splunk Enterprise Security, use CIM-compatible field names
  (`user`, `src`, `action`, `app`) so built-in correlation
  searches apply.
- HEC accepts batched events (newline-delimited JSON) for high-volume
  products.
- Splunk Cloud endpoint format:
  `https://http-inputs-<stack>.splunkcloud.com`.
