[Intercom](https://www.intercom.com) — customer messaging (chat,
product tours, support). Sync Ory identities into Intercom Contacts
so chat / support / lifecycle messaging is identity-aware.

## How it works

Ory Action on `registration.after` / `settings.after` POSTs to
`https://api.intercom.io/contacts` with bearer token auth.

## Notable

- **Identity verification**: HMAC-based `user_hash` in the Messenger
  prevents spoofing of authenticated chat sessions.
- Events API (`POST /events`) is the right path for tracking specific
  user actions, separate from contact updates.
