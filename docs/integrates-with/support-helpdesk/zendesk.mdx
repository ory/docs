[Zendesk](https://www.zendesk.com) — customer support platform. Sync
Ory identities into Zendesk Users via Ory Actions.

## How it works

Ory Action on `registration.after` (async) calls Zendesk's
`POST /api/v2/users.json` with HTTP Basic (email/token + API token)
or OAuth Bearer. `external_id` = Ory identity id.

## Notable

- Create Or Update User is idempotent on `external_id`.
- Rate limits per plan; high-volume signups need batching at scale.
- For Ory identity context **inside** Zendesk tickets, build a
  Zendesk App that calls the Ory admin API.
