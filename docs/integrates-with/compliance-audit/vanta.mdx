[Vanta](https://www.vanta.com) — automated security and compliance
platform (SOC 2, ISO 27001, HIPAA). Configure Vanta to pull identity
and access management evidence from Ory Network's admin API so
compliance evidence is collected automatically.

## Pattern

```
Vanta Custom Integration  →  Ory Admin API
        ↑                       (read-only Project API key)
        |
   Compliance Tests in Vanta evaluated against pulled data
```

1. Create an **Ory API key** (Project-scoped) with read access.
2. In Vanta → **Integrations** → **Add Custom Integration** (or via
   the Vanta API), configure an API poller against the Ory admin
   API on a schedule.

## Evidence to pull

| Endpoint | Vanta Test |
| --- | --- |
| `GET /admin/identities` | Personnel, Access Reviews, dormant accounts |
| `GET /admin/identities/{id}` | MFA enrollment, last-active timestamps |
| `GET /admin/clients` | Service accounts inventory |
| Project config | Password policy, session lifespan |
