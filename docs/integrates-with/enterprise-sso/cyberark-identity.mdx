[CyberArk Identity](https://www.cyberark.com/products/identity/) — SSO
and identity portion of the CyberArk platform, distinct from CyberArk's
PAM product. Exposes standard SAML 2.0 and OIDC endpoints for Ory Polis
to consume via the generic provider paths.

## Setup outline

1. In CyberArk Identity, create a **Custom SAML App** (or Custom OIDC
   App).
2. Configure the Polis SP metadata on the CyberArk app.
3. In Ory Network, configure the SSO connection via the generic SAML
   or generic OIDC walkthrough; paste CyberArk's metadata XML or
   issuer URL + client credentials.
4. Map released attributes (`email`, names, groups) to Ory traits.

PAM-specific REST integration is out of scope — this entry covers SSO
only.
