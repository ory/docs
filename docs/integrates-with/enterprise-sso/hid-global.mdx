[HID Global](https://www.hidglobal.com) unifies physical (badge
readers, smart cards, mobile credentials) and digital (SSO, MFA,
certificate-based authentication) identity. Common in government,
healthcare, financial services, and manufacturing.

## Setup outline

1. In the HID Authentication Service tenant, create a SAML 2.0 or
   OIDC application.
2. Paste Polis SP metadata into the HID app.
3. Define attribute release (email, names, groups).
4. In Ory Network, configure the SSO connection via the generic
   SAML or generic OIDC walkthrough.

Certificate-based auth (PIV / CAC / smart card) is enforced at HID;
Ory consumes the authenticated session.
