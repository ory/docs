[Keycloak](https://www.keycloak.org) — open-source CNCF-graduated IAM
server. Speaks SAML 2.0, OIDC, and OAuth 2.0. Common choice for
self-hosted IdP deployments.

## Setup outline (OIDC, recommended)

1. In Keycloak Admin → Clients → Create client (OpenID Connect,
   Confidential).
2. Set Valid redirect URI to Polis ACS URL.
3. Note Client ID + Secret.
4. Issuer URL pattern: `https://<keycloak-host>/realms/<realm-name>`
   — Polis discovers automatically via `/.well-known/openid-configuration`.
5. In Ory Network, configure via the generic OIDC walkthrough.

Group claims need an explicit Group Membership mapper on the client's
dedicated client scope.
