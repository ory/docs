[ForgeRock Access Management](https://www.pingidentity.com/en/platform/forgerock.html)
— now part of Ping Identity following the 2023 acquisition. Widely
deployed enterprise IAM supporting OIDC, OAuth 2.0, and SAML 2.0.
Ory Polis consumes ForgeRock as either an OIDC or SAML IdP.

## Setup outline

1. In ForgeRock AM, create a SAML 2.0 entity (under **Realms →
   Applications → Federation**) or an OAuth 2.0 client.
2. Configure Polis SP metadata on the ForgeRock app.
3. In Ory Network, configure the SSO connection via the generic SAML
   or generic OIDC walkthrough.

Polis is SP-initiated only.
