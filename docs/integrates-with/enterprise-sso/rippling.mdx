Configure Rippling — the unified workforce platform combining HR, IT, and
Finance — as a SAML 2.0 Identity Provider into Ory Polis. A natural fit for
organizations already using Rippling for HR/IT-driven access lifecycle, where
onboard, offboard, and role-change events trigger access changes automatically.
