[IBM Security Verify](https://www.ibm.com/products/verify-identity) —
IBM's cloud-delivered IAM platform. OIDC + SAML 2.0 endpoints
consumed by Ory Polis via the generic provider paths.

## Setup outline

1. In IBM Security Verify Admin → Applications → Add application,
   pick **Custom SAML 2.0** or **Custom OIDC**.
2. Configure with Polis SP details.
3. Map outbound attributes (email, names, groups).
4. In Ory Network, configure the SSO connection via the generic
   SAML or OIDC walkthrough.

Both Verify Cloud (SaaS) and Verify Access (on-prem) use the same
federation flows; risk-based access policies stay on the IBM side.
