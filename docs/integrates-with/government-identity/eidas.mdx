[eIDAS](https://digital-strategy.ec.europa.eu/en/policies/eidas-regulation)
— the EU regulation enabling cross-border use of national electronic IDs
(German Personalausweis, Italian SPID, Belgian eID, French
FranceConnect, Spanish DNI, etc.).

## Integration path

Use a commercial eIDAS broker exposing eIDAS as standard OIDC;
configure as a generic OIDC provider in Kratos. Brokers handle the
per-country eIDAS node integration.

Common brokers: Signicat (pan-EU), Criipto, IDnow, yes.com (DE).

## Assurance levels

eIDAS defines three Levels of Assurance — map them to Ory session AAL:

| eIDAS LoA | Ory AAL |
| --- | --- |
| Low | aal1 |
| Substantial | aal1 / aal2 |
| High | aal2 |

Broker reports LoA in the ID token; Jsonnet maps to
`metadata_public.eidas.loa` for downstream policy.
