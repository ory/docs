[Microsoft Dynamics 365](https://www.microsoft.com/en-us/dynamics-365)
— enterprise CRM and ERP. Sync Ory identities to Dynamics 365 Contact
records via the Dataverse Web API (OData v4) using an Ory Actions
webhook.

## How it works

Ory Action on `registration.after` (async) → handler upserts a Contact
in Dynamics 365 via the Dataverse Web API
(`https://<env>.crm.dynamics.com/api/data/v9.2`), keyed by an
`ory_identity_id` alternate key. Auth via Entra ID OAuth2 client
credentials.

## Notable

- Dataverse environments are per-region — pin the endpoint host.
- The `ory_identity_id` alternate key must be created in Dataverse
  before first run.
- Async with `response.ignore: true` — Dataverse availability never
  blocks Ory user flows.
