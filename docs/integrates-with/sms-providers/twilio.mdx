[Twilio](https://www.twilio.com) — the most widely deployed cloud SMS
platform. Canonical example in the Ory SMS courier documentation.

## Quick reference

| Setting | Value |
| --- | --- |
| URL | `https://api.twilio.com/2010-04-01/Accounts/{$ACCOUNT_SID}/Messages.json` |
| Method | `POST` |
| Content-Type | `application/x-www-form-urlencoded` (**required** — Twilio rejects JSON) |
| Auth | HTTP Basic (`Account SID` : `Auth Token`) |
| Body | URL-encoded `To`, `From`, `Body` |

## Notable

- Trial accounts can only send to verified numbers; production
  delivery requires a paid account.
- Twilio offers **Test Credentials** (separate SID/Token) that
  simulate sends without delivery or charges — useful for CI.
- For production, prefer a Twilio **Messaging Service** (sender
  pool, country-specific routing, compliance) — replace `From=...`
  with `MessagingServiceSid=MGxxxx...`.
