[Vonage](https://www.vonage.com) (formerly Nexmo) — global cloud
communications platform with strong EMEA / APAC carrier coverage.
Common alternative to Twilio for products with significant
international user bases.

## Quick reference

| Setting | Value |
| --- | --- |
| URL | `https://rest.nexmo.com/sms/json` |
| Method | `POST` |
| Auth | API key + secret in the **request body** (not headers) |
| Body | JSON with `api_key`, `api_secret`, `from`, `to`, `text` |

Vonage authenticates by including `api_key` and `api_secret` in the
JSON body, so the Kratos courier `auth:` block stays empty —
credentials live in the Jsonnet template.

## Notable

- Vonage accepts E.164 with or without the leading `+`; strip the `+`
  in Jsonnet if a country rejects the `+` form.
- Alphanumeric sender ID (up to 11 chars) supported in many countries;
  not allowed in US / Canada (registered number required).
- Newer Messages API (`api.nexmo.com/v1/messages`) available with
  JWT/header auth; the REST SMS API above is simpler for SMS-only use.
