[Plivo](https://www.plivo.com) — cloud-communications platform with
direct carrier connections in 190+ countries. Common pick when SMS unit
economics matter.

## Quick reference

| Setting | Value |
| --- | --- |
| URL | `https://api.plivo.com/v1/Account/{$AUTH_ID}/Message/` |
| Method | `POST` |
| Auth | HTTP Basic (`Auth ID` : `Auth Token`) |
| Body | JSON with `src` (E.164), `dst` (E.164), `text` |

## Notable

- Plivo accepts E.164 directly (no `+` stripping).
- For production, consider a Plivo **Powerpack** (number pool with
  intelligent routing) — replace `src` with `powerpack_uuid` in
  Jsonnet.
