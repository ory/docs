[MessageBird](https://www.messagebird.com) (rebranded as Bird) — European
cloud-communications platform with strong EU/APAC coverage. Common
alternative to Twilio for products with EU regulatory or data-residency
concerns.

## Quick reference

| Setting | Value |
| --- | --- |
| URL | `https://rest.messagebird.com/messages` |
| Method | `POST` |
| Auth | header `Authorization: AccessKey {$API_KEY}` |
| Body | JSON with `originator`, `recipients[]` (MSISDN, **no `+` prefix**), `body` |

## Notable

- MessageBird expects **MSISDN** format (no leading `+`). The Jsonnet
  body must strip the `+` from Ory's E.164 recipient.
- Originator can be a MessageBird virtual number or an alphanumeric
  sender ID (up to 11 chars; not allowed in US/Canada/China).
- Optional `reportUrl` in the body enables real-time delivery
  callbacks.
