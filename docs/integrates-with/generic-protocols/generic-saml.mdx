Any SAML 2.0-compliant Identity Provider is configured into Ory Polis as the SAML Service Provider — no custom code required. See

Configure any SAML 2.0-compliant Identity Provider with Ory. SAML support in Ory
is provided by Ory Polis (formerly SAML Jackson) acting as the SAML Service
Provider. Ory Identities (Kratos) does not natively speak SAML.
