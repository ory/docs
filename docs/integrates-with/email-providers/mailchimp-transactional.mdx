[Mailchimp Transactional](https://mailchimp.com/developer/transactional/)
(formerly Mandrill) is the paid transactional-email add-on to Mailchimp. It
fits well when you already use Mailchimp for marketing.

- See [Mailchimp integration with Ory Actions](../../actions/integrations/mailchimp.mdx)
  for the primary documentation.
- For the raw SMTP recipe, see
  [Sending emails over SMTP](../../kratos/emails-sms/01_sending-emails-smtp.mdx#mailchimp-transactional-mandrill).
