[SparkPost](https://www.sparkpost.com) (a Bird/MessageBird company) —
enterprise-grade transactional email platform with per-region US / EU
hosting.

## Quick reference

| Setting | Value |
| --- | --- |
| SMTP host (US) | `smtp.sparkpostmail.com` |
| SMTP host (EU) | `smtp.eu.sparkpost.com` |
| SMTP port | `587` (STARTTLS) |
| SMTP login | literal string `SMTP_Injection` (correct for all customers) |
| SMTP password | SparkPost API key |
| REST API base (US) | `https://api.sparkpost.com/api/v1` |
| REST API base (EU) | `https://api.eu.sparkpost.com/api/v1` |

Connection URI:

```
smtps://SMTP_Injection:<api-key>@smtp.sparkpostmail.com:587
```

## Notable

- SparkPost separates US and EU accounts — pick data residency before
  integration; account migrations are not trivial.
- API key needs the **Send via SMTP** permission for the SMTP path.
- Stored templates are available via the API; not recommended for Ory
  identity emails — Ory renders templates by default.
