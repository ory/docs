[Brevo](https://www.brevo.com) (formerly Sendinblue) — European email
platform with strong GDPR posture. Common when EU-hosted email
infrastructure is required.

## Quick reference

| Setting | Value |
| --- | --- |
| SMTP host | `smtp-relay.brevo.com` |
| SMTP port | `587` (STARTTLS) or `465` (TLS) |
| SMTP user | Your Brevo account email |
| SMTP password | Brevo SMTP key (under **SMTP & API → SMTP**) |
| HTTP endpoint | `POST https://api.brevo.com/v3/smtp/email` with `api-key` header |

Connection URI:

```
smtps://<smtp_login>:<smtp_key>@smtp-relay.brevo.com:587
```

## Notable

- Brevo SMTP keys are **distinct from API keys** — don't confuse them.
- Free tier rate-limits SMTP; identity flows at scale need a
  paid plan.
- DKIM and Return-Path alignment with the From domain is required for
  Gmail / Microsoft deliverability.
