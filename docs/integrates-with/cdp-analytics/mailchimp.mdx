Sync new Ory Network users into a [Mailchimp](https://www.mailchimp.com/)
audience via Ory Actions so registrations automatically appear in the
marketing CRM.

Distinct from the
[Mailchimp Transactional (Mandrill) integration](../email-providers/mailchimp-transactional.mdx)
— that sends transactional emails; this one manages marketing audiences.

## How it works

Ory Action on `registration.after` (async) POSTs to Mailchimp's
Lists/Audiences API:

- URL:
  `https://{dc}.api.mailchimp.com/3.0/lists/{list_id}/members`.
- Auth: HTTP Basic with username `anystring` and password
  `{api_key}`.
- Body Jsonnet maps `identity.traits.email` and
  `identity.traits.name` into Mailchimp merge fields.

The Mailchimp Marketing API key is **distinct from** the Mandrill key
used for the transactional integration.
