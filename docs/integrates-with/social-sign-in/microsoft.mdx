```mdx-code-block
Microsoft is configured as a social sign-in provider in the Ory Console or via the Ory CLI — no webhook code required. See
[Add Microsoft as a social sign-in provider in Ory](../../kratos/social-signin/20_microsoft.mdx) for the full guide.
</Admonition>
```

Add Microsoft as a social sign-in provider in Ory Network. The Microsoft identity platform covers both consumer Microsoft accounts
(Outlook, Xbox, Hotmail) and organizational accounts (Microsoft Entra ID / Azure AD), so the integration works for both B2C and
B2B flows.
