[Epic Games](https://dev.epicgames.com/portal) provides OAuth 2.0 / OIDC
sign-in for the Epic Games Store and Fortnite player base. Configure as
a social sign-in provider in Ory Network for gaming, community, and
companion-app use cases targeting Epic users.

## Setup

The full walkthrough lives at
[ory/integrates/social-sign-in/epic-games](https://github.com/ory/integrates/tree/main/social-sign-in/epic-games).
Short version:

1. Create an Epic Games product at
   [dev.epicgames.com/portal](https://dev.epicgames.com/portal).
2. Enable the **Account Service** and add the Ory redirect URI.
3. Configure as a generic OIDC provider in Kratos with:
   - Issuer URL: `https://api.epicgames.dev/epic/oauth/v2`
   - Authorization endpoint: `https://www.epicgames.com/id/authorize`
   - Scopes: `basic_profile` plus any optional approved scopes.
4. Map the OIDC claims (`sub`, `display_name`, `preferred_language`)
   to Ory identity traits via Jsonnet.

Epic Games application review is required before `basic_profile` and
adjacent scopes are released — plan for review lead time.
