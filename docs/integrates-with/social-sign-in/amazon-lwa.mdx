```mdx-code-block
Amazon (Login with Amazon) is configured as an OpenID Connect social sign-in provider in the Ory Console or via the Ory CLI — no webhook code required. See
[Add Amazon as a social sign-in provider in Ory](../../kratos/social-signin/99_amazon.mdx) for the full guide.
</Admonition>
```

Add Amazon (Login with Amazon, LWA) as a social sign-in provider in Ory Network. Useful for consumer products targeting the Amazon ecosystem (Alexa skills, FireOS apps, retail-adjacent flows) where users are already signed in to Amazon.
