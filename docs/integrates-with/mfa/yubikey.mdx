YubiKey is a hardware security key from Yubico that supports WebAuthn / FIDO2 / U2F. Ory natively supports WebAuthn / FIDO2 as a
multi-factor authentication method, so YubiKey registration and use is built-in — no integration code required, just enable
WebAuthn in your project's MFA configuration.
