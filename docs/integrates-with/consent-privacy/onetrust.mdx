[OneTrust](https://www.onetrust.com) — privacy and consent management
platform (GDPR, CCPA, LGPD, HIPAA). Sync Ory identities with
OneTrust profiles and consume OneTrust consent webhooks to surface
consent state on Ory identities.

## How it works

**Outbound** — Ory Action on `registration.after` → OneTrust Profile
API to create/update profile keyed by Ory identity id.

**Inbound** — OneTrust webhook on consent changes → handler verifies
HMAC-SHA256 → PATCHes `metadata_public.consent.onetrust` on the
matching Ory identity.

**DSAR fulfillment** — OneTrust forwards DSARs to your handler; map
to `GET` (Article 15) / `DELETE` (Article 17) on the Ory admin API.
