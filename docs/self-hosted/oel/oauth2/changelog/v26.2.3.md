## v26.2.3

### Remove CGO requirement from Ory by switching to modernc.org/sqlite

Migrates SQLite support from `mattn/go-sqlite3` (CGO) to `modernc.org/sqlite` (pure Go) and fixes several timestamp comparison
bugs that caused incorrect pagination results. This migration is fully backwards compatible. Going forward, the `-tags sqlite` tag
is no longer required to compile Ory.
