We're happy to announce the release of our new product: Ory Talos - API Keys. Ory Talos 
secures automated workflows by turning static API keys into short-lived capability tokens 
for non-human identities. Using token derivation and Macaroon delegation, it enforces 
strict least-privilege guardrails.
